Stock PDF Reporter

Professional Technical Analysis Reports for Chinese A-Share Stocks

One command. Beautiful PDF. No API key.

Python 3.8+ MIT License Free & Open Source
Get on GitHub Order Report

Features

  • Real-time A-share data via Sina
  • MA/5/10/20/60, MACD, RSI, Bollinger
  • Price chart + volume + indicators
  • Auto technical signal analysis
  • Beautiful 5-page professional PDF

Quick Start

pip install akshare fpdf2 matplotlib pandas numpy

python stock_report.py 600519

python stock_report.py 300750 --days 120

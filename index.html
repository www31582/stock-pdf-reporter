Stock PDF Reporter

Professional Technical Analysis Reports for Chinese A-Share Stocks

One command. Beautiful PDF. No API key needed.

Python 3.8+ MIT License Free & Open Source
GitHub Repo

Features

  • Real-time A-share data via Sina Finance
  • MA5/10/20/60, MACD, RSI, Bollinger Bands
  • Price chart + volume + technical indicators
  • Auto-generated technical signal analysis
  • Beautiful 5-page professional PDF

Quick Start

pip install akshare fpdf2 matplotlib pandas numpy

# Maotai 60-day report
python stock_report.py 600519

# Ping An 120-day report
python stock_report.py 601318 --days 120
